today we learned about... how to add things to my online version of my website

  1. Open my terminal and navigate to my folder by typing cd ./desktop/c27dk.github.io
  2. If I want to see if I added anything I can type git status and it can show me my changes
  3. to add what I made I type git add .
  4. Then I commit my changes and this means adding a little note that marks what I did so I saw git commmit -m "whatever I did"
  5. To finish it off and put my changes online I type git push
